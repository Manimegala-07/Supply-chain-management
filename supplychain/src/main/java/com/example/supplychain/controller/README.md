# Supply Chain Management System

## 📌 Description
A Spring Boot based Supply Chain Management system with product, inventory,
order, supplier, and stock history modules.

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL
- HTML, CSS

## ✨ Features
- Product management
- Inventory tracking
- Stock history logs
- Order processing

## 🚀 How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run the Spring Boot application
