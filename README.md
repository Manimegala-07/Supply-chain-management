# Supply Chain Management System

A backend-driven Supply Chain Management system built using **Java Spring Boot**.  
The system manages products, suppliers, customers, orders, and inventory while tracking stock changes through a structured API-based architecture.

This project demonstrates **RESTful backend development, database integration, and inventory tracking logic** used in real-world enterprise systems.

---

## Tech Stack

Backend
- Java
- Spring Boot
- Spring Data JPA
- REST APIs

Database
- MySQL

Frontend
- HTML
- CSS
- JavaScript

Tools
- Postman (API Testing)
- Maven
- Git & GitHub

---

## Key Features

### Product Management
- Add new products
- Retrieve all products
- Get product by ID
- Check low stock products
- View stock levels

### Supplier Management
- Add suppliers
- View supplier details
- Manage supplier records

### Customer Management
- Create customers
- Retrieve customer information

### Order Processing
- Create orders
- Track order details
- Link orders with customers and products

### Inventory Management
- Track product stock
- Update inventory levels
- Calculate total inventory value

### Stock History Tracking
- Log inventory changes
- Record actions such as:
  - STOCK_IN
  - STOCK_OUT
- Store timestamps for audit tracking
